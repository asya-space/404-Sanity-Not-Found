## About

This project began as a simple challenge: build a responsive meme slider using HTML and CSS only, without JavaScript or UI frameworks.

Rather than creating a standard implementation, I used it as an opportunity to deepen my understanding of CSS and experiment with building an interactive UI using native browser features.

The result is a fully responsive slider with a custom visual style, hand-picked memes, and personalized captions that make the project feel more engaging than a typical practice assignment.

## Tech Stack
 - HTML5
 - CSS3
 - Responsive Design
 - CSS-only interaction (no JavaScript)

## Features

 - 🎭 CSS-only meme slider
 - 📱 Responsive layout for mobile, tablet, and desktop
 - 🎨 Custom styling and themed visual design
 - 🖼️ Carefully selected memes and captions
 - 🚫 No frameworks or JavaScript

## Why this project?

Whenever possible, I use my practice tasks as an excuse to experiment, refine my skills, and build something with a bit more personality.

This project was primarily about strengthening my CSS fundamentals while creating an interface that feels polished, enjoyable, and fun to explore.
